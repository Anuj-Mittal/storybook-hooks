# `useCounter`

React state hook that tracks a numeric value.

## Usage

```jsx
import { useCounter } from "./useCounter";

interface CounterProps {
  initialValue: number;
  min?: number | null;
  max?: number | null;
}
const Comp = ({ initialValue, min, max }: CounterProps) => {
  const [current, actions] = useCounter(initialValue, min, max);

  return (
    <div>
      <h3>Min : {min}</h3>
      <h3>Max : {max}</h3>
      <button onClick={() => actions.dec()}>Decrement</button>
      {current}
      <button onClick={() => actions.inc()}>Increment</button>
      <button onClick={() => actions.reset()}>Reset Counter</button>
    </div>
  );
};
```

## API

```ts
const [ current, { inc, dec, get, set, reset } ] = useCounter(initialValue: number, max: number | null = null, min: number | null = null);
```

- `current` - current counter value;
- `get(): number` - getter of current counter value;
- `inc(delta: number): void` - increment current value;
- `dec(delta: number): void` - decrement current value;
- `set(value: number): void` - set arbitrary value;
- `reset(value: number): void` - as the `set`, but also will assign value by reference to the `initial` parameter;
